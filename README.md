一、游戏玩法

鼠标控制大鱼游动，吃到果实去喂小鱼，黄色果实10分，蓝色果实20分，如果小鱼在一段时间内吃不到果实，身体颜色会慢慢变浅，变为全白后，小鱼死亡，游戏结束。
